# Building Bills Management System

سیستم مدیریت مصارف برق و آب ساختمان

## ویژگی‌ها

- مدیریت واحدها (افزودن/ویرایش/حذف)
- ثبت قرائت ماهانه (برق عمومی، آب عمومی، مترهای فرعی برق)
- محاسبه خودکار هزینه‌ها با نرخ‌های قابل تنظیم
- تقسیم هزینه برق مشاع به صورت نسبتی بر اساس مصرف هر واحد
- تقسیم آب به صورت مساوی بین واحدها
- صدور قبض برای هر واحد
- پیگیری بدهی دوره‌های قبل
- آپلود عکس کنتور برای هر واحد
- بک‌آپ و بازیابی داده‌ها (JSON)
- پرینت قبض

## تغییرات نسخه جدید

1. **اصلاح محاسبه بدهی گذشته**: بدهی دوره‌های قبل فقط شامل `currentCost` (جمع دوره جاری) می‌شود، نه `totalCost` (جمع دوره جاری + بدهی گذشته). این مانع از رشد نمایی بدهی می‌شود.

2. **آپلود عکس کنتور**: امکان آپلود عکس کنتور برق برای هر واحد در زمان ثبت قرائت. عکس‌ها در نمایش محاسبه و قبض قابل مشاهده و بزرگ‌نمایی هستند.

3. **بک‌آپ و بازیابی JSON**: امکان خروجی گرفتن از تمام داده‌ها در یک فایل JSON و بازیابی آن‌ها.

## نحوه استفاده

1. فایل `index.html` را در مرورگر باز کنید.
2. تنظیمات (نرخ برق و آب، مشخصات ساختمان) را وارد کنید.
3. واحدها را تعریف کنید.
4. هر ماه قرائت جدید ثبت کنید.
5. قبض‌ها را مشاهده و پرینت کنید.

## ذخیره‌سازی

تمام داده‌ها در `localStorage` مرورگر ذخیره می‌شوند.
